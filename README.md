# Trello Fast API Service

The Trello Fast API is a task manager which allows managers to add projects and tasks for the respective users and track their progress.

The Trello Fast API service is divided into three sections:

## User Management
A service which allows admins to register new users and new users to login.

## Project Management
A service for creating a new project

## Task Management
A service which allows admins to create new tasks and allows users to update their existing tags.
